# DiamonMiner Dashboard (Prototype)

This is a simple prototype static dashboard for **DiamonMiner** with token **$DIA**.

Features:
- Card-style dashboard layout
- Animated glow diamond logo
- Tailwind Play CDN used for quick styling (no build step)
- Includes placeholder widgets and quick actions

How to use:
1. Unzip `diamondminer_dashboard.zip`.
2. Open `index.html` in a browser.

Notes:
- This is a front-end prototype only (no backend, no wallet integration).
- The "Create your $DIA Token" section has been removed as requested.
- Social link: https://x.com/Diamonminer
